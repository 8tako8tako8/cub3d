# cub3D

## Overview
42Tokyoの課題。
ray castingという技術を用いて3D空間を実装する。
武器や敵は未実装。
map内にある樽を見つける。

## Requirement
- mac Catalina 10.15.7
or
- XUbuntu 18.04.5

## Usage
```
git clone ...
cd cub3d
make
./cub3D map_file/meiro.cub
```
スタートポジションのbmp作成
```
./cub3D map_file/meiro.cub --save
```
